# vue3-cli-bidding
## 利用vue-cli工具升级vue2到vue3
+ 1、确保vue-cli版本 >= 4.3.1
```
vue -V
```
+ 2、用vue-cli 创建项目
```
vue create vue3-cli-bidding
```
+ 3、升级vue版本
```
vue add vue-next
```
## Project setup
```
yarn install
```
